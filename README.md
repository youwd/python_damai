## 准备工具
* Python3.6
* Selenium
* chrome浏览器，需要下载ChromeDriver包及配置  http://chromedriver.storage.googleapis.com/index.html 
* firefox浏览器，需要下载geckodriver包及配置  https://github.com/mozilla/geckodriver/releases/tag/v0.24.0  下载好放到python对应文件夹下
